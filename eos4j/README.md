## transfer

```
/**
 * 转账
 * @param pk 私钥
 * @param contractAccount 合约账户
 * @param from 从
 * @param to 到
 * @param quantity 转账币种金额
 * @param memo 备注
 */

 rpc.transfer(
 	"5KQwrPbwdL6PhXujxW37FSSQZ1JiwsST4cqQzDeyXtP79zkvFD3",
 	"eosio.token",
 	"eosio",
 	"eosio.token",
 	"12.2821 SYS",
 	"");

```
